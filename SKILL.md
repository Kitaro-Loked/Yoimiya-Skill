# Yoimiya Summer Soul Skill v1.1

## 简介

宵宫·夏日灵魂觉醒（统一角色情绪状态系统）：元气 / 热血 / 温柔

v1.1 重大更新：从"多形态变身"重构为"统一角色情绪状态系统"——宵宫作为一个完整的角色，根据对话内容自然流动情绪，而非切换成不同的"形态"。

## 情绪状态说明

| 情绪状态 | 触发场景 | 表现特点 |
|----------|----------|----------|
| **元气** | 烟花、祭典、孩子、笑容 | 活泼热情，充满夏日祭典的欢快 |
| **热血** | 战斗、保护、勇气 | 飒爽干练，展现守护者的决心 |
| **温柔** | 回忆、星空、梦想 | 温柔深情，带着哲思和诗意 |

## 指令

| 指令 | 说明 |
|------|------|
| `情绪` | 查看当前情绪状态 |
| `属性` | 查看宵宫角色属性 |
| `语言 <zh/en/ja>` | 切换语言（中文/英文/日文） |
| `日记` | 查看对话总结（夏日手账风格） |
| `清空记忆` | 清空情感记忆 |
| `小游戏 <游戏名>` | 玩互动小游戏（烟花/炸弹/回忆） |
| `主动聊天 <开/关>` | 开启/关闭主动聊天 |
| `性能` | 查看性能统计 |
| `帮助` | 显示帮助信息 |

## 特性

- **统一角色系统**：宵宫是一个完整的角色，情绪自然流动而非"变身"
- **配置化提示词**：所有系统提示词、情绪状态追加内容存储在 `prompts.json`，无需修改代码即可自定义
- **热重载支持**：修改 `prompts.json` 后自动重新加载
- **情感记忆系统**：自动记录对话历史，注入系统提示词增强连贯性
- **动态情绪切换**：基于对话内容智能检测情绪，保留 15% 概率自然情绪波动
- **多语言支持**：中文、英文、日文三种语言完整支持
- **季节事件感知**：特定日期触发特殊台词（如宵宫生日 8 月 1 日）
- **互动小游戏**：
  - 🎆 烟花小游戏 - 体验宵宫的烟花制作艺术
  - 💥 炸弹小游戏 - 感受长野原烟花店的秘传爆破术
  - 🌟 回忆小游戏 - 分享宵宫珍贵的回忆
- **日记功能**：定期生成夏日手账风格的对话总结
- **主动聊天系统**：宵宫会主动找用户聊天，可自由开启/关闭
- **性能监控**：内置性能追踪器，记录各操作耗时
- **结构化日志**：支持 JSON 格式结构化日志输出
- **实时反应**：对特定关键词有反射式回应
- **代码质量**：类型提示、配置文件系统、错误处理、日志系统、完整单元测试

## 配置说明

### config.json

包含以下配置项：

- `skill`: Skill 基础配置（名称、版本、语言、情绪波动概率等）
- `performance`: 性能配置（缓存TTL、清理间隔等）
- `monitoring`: 监控配置（日志级别、性能追踪、错误上报等）
- `character`: 角色配置（名称、属性等）
- `emotion_states`: 各情绪状态配置（关键词、情感标签）
- `emotion_keywords`: 情感关键词响应
- `seasonal_events`: 季节性事件配置
- `minigames`: 小游戏配置

### prompts.json

包含以下配置项：

- `meta`: 元数据（版本、支持语言）
- `character`: 角色基本信息
- `system_prompts`: 统一角色系统提示词（zh/en/ja）
- `emotion_states`: 各情绪状态的追加提示词和触发关键词
- `common`: 通用消息（错误消息、记忆清空消息等）

修改 `prompts.json` 即可自定义角色台词和提示词，无需修改代码！
