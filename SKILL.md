# Yoimiya Summer Soul Skill v1.0

## 简介

宵宫·夏日灵魂觉醒（究极情感沉浸版）：夏祭·元气 / 琉金·热血 / 幻梦·温柔

v1.0 首发：全面配置化提示词，所有文本从代码剥离至 `prompts.json`，支持热重载和多语言扩展。

## 形态说明

| 形态 | 特点 | 属性倾向 |
|------|------|----------|
| **夏祭** | 元气活泼的烟花女王，稻妻孩子们心目中的英雄，热爱祭典和烟花 | 高速度、高暴击 |
| **琉金** | 英姿飒爽的弓箭战士，用火焰与勇气守护珍视的一切 | 高速度、高攻击 |
| **幻梦** | 温柔深情的星空哲人，在寂静夜晚思考烟花与人生 | 均衡型、高温柔 |

## 指令

| 指令 | 说明 |
|------|------|
| `切换形态 <形态名>` | 手动切换形态，支持：夏祭 / 琉金 / 幻梦 |
| `形态属性` | 查看当前形态的属性面板 |
| `语言 <zh/en/ja>` | 切换语言（中文/英文/日文） |
| `日记` | 查看对话总结（夏日手账风格） |
| `清空记忆` | 清空情感记忆 |
| `小游戏 <游戏名>` | 玩互动小游戏（烟花/炸弹/回忆） |
| `主动聊天 <开/关>` | 开启/关闭主动聊天系统 |
| `帮助` | 显示帮助信息 |

## 特性

- **配置化提示词**：所有系统提示词、变身台词、切换回复存储在 `prompts.json`，无需修改代码即可自定义
- **热重载支持**：修改 `prompts.json` 后自动重新加载（或调用 `reload_prompts()`）
- **情感记忆系统**：自动记录对话历史，注入系统提示词增强连贯性
- **动态形态切换**：基于对话内容智能推荐形态，保留 15% 概率自动变身
- **多语言支持**：中文、英文、日文三种语言完整支持
- **形态属性系统**：每种形态拥有独立的属性面板（防御/生命/速度/攻击/暴击）
- **季节事件感知**：特定日期触发特殊台词（如宵宫生日 8 月 1 日）
- **互动小游戏**：
  - 🎆 烟花小游戏 - 体验宵宫的烟花制作艺术
  - 💥 炸弹小游戏 - 感受长野原烟花店的秘传爆破术
  - 🌟 回忆小游戏 - 分享宵宫珍贵的回忆
- **日记功能**：定期生成夏日手账风格的对话总结
- **主动聊天系统**：宵宫会主动找用户聊天，可自由开启/关闭
- **实时反应**：对特定关键词有反射式回应
- **代码质量**：类型提示、配置文件系统、错误处理、日志系统、策略模式、完整单元测试

## 配置说明

### config.json

包含以下配置项：

- `skill`: Skill 基础配置（名称、版本、默认形态、语言等）
- `forms`: 各形态配置（属性、关键词、情感标签）
- `emotion_keywords`: 形态专属情感关键词响应
- `seasonal_events`: 季节性事件配置
- `minigames`: 小游戏配置

### prompts.json

包含以下配置项：

- `meta`: 元数据（版本、支持语言）
- `forms`: 各形态的提示词（系统提示词、变身台词、切换回复），每种支持 zh/en/ja
- `common`: 通用消息（错误消息、记忆清空消息等）

修改 `prompts.json` 即可自定义角色台词和提示词，无需修改代码！
